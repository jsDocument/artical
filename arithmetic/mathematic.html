  • 排列、组合

    • 加法：完成一件事有n类办法，方法数等于种类方法的和
    • 乘法：完成一件事有n步，完成这件事等于各步方法相乘

  • 排列：n个不同元素中取出m个元素，按一定的顺序排成一列An~m = n!/(n-m)!

    • n! = n*(n-1)!
    • (n+1)! = (n+1)*n!
    • n * n! = [(n+1)-1] * n! = (n+1)*n! - n! = (n+1)! - n!
    • n/(n+1)! = n+1-1/(n+1)! = n+1/(n+1)! - 1/(n+1)! = 1/n! - 1/(n+1)!

  • 组合：n个元素中任取m个元素并组成一组；cn~m = An~m/Am~m = n!/m!*(n-m)!
