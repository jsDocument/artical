# 集群---路由、分片、副本集、配置服务

  • mongos 是一个请求分发中心，负责把对应数据请求转发到对应shard服务器上

  • config server 存储所有数据库元信息(路由、分片)的配置，mongos本身没有存储，只是缓存在内存里

  • shard 数据库拆分，将其分散在不同机器上的过程（主节点、副节点、仲裁）

  • replica set shard的备份

  • --configdb
